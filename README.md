## A simple survey form for shy guys who want to ask which gifts do their friend prefer.
1. Analyze (0.5 hours)
  - Read the requirement and list needed features
  - Choose design and colors
2. Coding (2.5 hours)
  - Clone forked repo
  - Install packages
  - Create ticket lists
    - List view
    - Filter by status
    - Paging
    - Create new ticket button
    - Navigate to ticket detail
    - Handle race condition with Skeleton
    - Responsive
  - Create ticket detail
    - Back to list
    - Create new ticket
    - Assign user
    - Unassign user
    - Mark as complete
    - Mark as uncomplete
3. Testing (0.5 hours)
  - Testcases:
    - Filter by "Complete" status 
    - Snackbar show "Ticket added!" on create new ticket.

## Techstack: NextJS, Tailwind, Material UI, EmailJS
